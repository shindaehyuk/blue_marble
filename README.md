# 부름마블 — 일꾼을 세워라! 🎲

2026.07.17 총무부 ✕ 친교부 연합교제용 부루마블 스타일 웹 보드게임입니다. (네이비 + 골드 테마)

## 실행 방법

- **웹/모바일**: https://shindaehyuk.github.io/blue_marble/ 접속 (핸드폰에서도 바로 실행됩니다)
- **로컬**: 별도 설치 없이 `index.html`을 브라우저로 열면 바로 실행됩니다.

개발 브랜치에 푸시하면 GitHub Actions가 `gh-pages` 브랜치로 미러링해 사이트가 자동 갱신됩니다.

## 구성

| 파일 | 설명 |
|---|---|
| `index.html` | 인트로(게임 시작) → 보드판 → 칸 정보 / 일꾼 세우기 / 주사위 모달 |
| `golden_key.html` | 황금열쇠 뽑기 페이지 (보드로 돌아가기 버튼 포함) |
| `assets/` | 각 칸 이미지(PPT 추출) · `logo.svg`(임시 로고) — **`assets/logo.png`를 넣으면 원본 로고로 자동 교체** |

## 기능

- **4팀 진행** — 강아지팀 🐶 / 고양이팀 🐱 / 토끼팀 🐰 / 여우팀 🦊
- **보드 24칸** — 본당 로비(출발) · 사망의 골짜기 · 총무부 · 바자회 + 부서 16칸 + 황금열쇠 3칸 + 상급 1칸
- **팀 구역** — 부서 16칸을 4칸씩 4팀 구역으로 나누어 팀 색 테두리 표시, 홈팀이 일꾼을 세우면 **통행료 ×2** 배지
- **말 이동** — 동물 캐릭터 말을 마우스(터치)로 자유롭게 드래그
- **칸 정보 모달** — 칸 클릭 시 부서별 미니게임 방법/승리 조건 표시
- **왼쪽 사이드 메뉴** — 게임 규칙 보기 모달 · 팀 이름 수정하기(localStorage 저장, 전체 화면 즉시 반영)
- **일꾼 세우기** — 부서 16칸에만 가능 (우측 상단 `+` 버튼 → 팀 선택 → 배치/제거)
- **주사위 모달** — 주사위 2개 굴리기, 구르는 소리(Web Audio), 더블 표시
- **황금열쇠** — 황금열쇠 칸/버튼에서 뽑기 페이지로 이동, 돌아오기 버튼 제공
- 말 위치·일꾼 배치는 브라우저(localStorage)에 자동 저장됩니다

## 내용 수정

- 칸 이름/설명/미니게임: `index.html`의 `TILES` 배열
- 팀 이름/색상/캐릭터: `index.html`의 `TEAMS` 배열
- 황금열쇠 카드 내용: `golden_key.html`의 `GOLDEN_KEYS` 배열
